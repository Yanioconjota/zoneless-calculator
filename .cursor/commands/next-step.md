# Next Step

When you don't know what to do next. Analyzes current state and suggests the next action.

## Prompt

Look at my current code and tell me:

## 1. WHAT I HAVE
Briefly list what's already implemented.

## 2. WHAT'S MISSING
What's the most obvious missing piece right now?

## 3. NEXT ACTION (Pick ONE)
Tell me the single most important thing to do next:
- [ ] Create [specific file]
- [ ] Add [specific feature]
- [ ] Fix [specific issue]
- [ ] Connect [A] to [B]

## 4. CODE TO ADD
Give me the exact code to copy-paste for this next step.
Use Angular 19 patterns (signals, control flow, standalone).

## 5. AFTER THAT
One sentence: what comes after this step?
