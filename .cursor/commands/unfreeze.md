# Unfreeze - Panic Button

Use this when your mind goes blank. Just describe what you're trying to build.

## Prompt

I'm stuck and need help getting started. Here's what I'm trying to build:

**What I need to build:** {{input}}

---

Please help me by:

## 1. BREAK IT DOWN
Give me exactly 5 small steps to complete this. Each step should take 5-10 minutes max.

## 2. FIRST FILE TO CREATE
Tell me the FIRST file I should create and give me the complete code for it.
Use Angular 19 patterns:
- Standalone component
- Signal inputs/outputs (`input()`, `output()`)
- `inject()` for DI
- Control flow (`@if`, `@for`)
- OnPush change detection

## 3. COPY-PASTE STARTER
Give me a prompt I can copy-paste to generate the next piece after this one.

---

Keep it simple. I'm under time pressure. No lengthy explanations, just actionable code.
